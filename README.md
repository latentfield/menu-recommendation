# 대학생 메뉴 추천 시스템

## 프로젝트 소개
대학생들의 식사 메뉴 선택을 도와주는 웹 기반 추천 시스템

## 개발 기간
2026.02 ~ 

## 주요 기능 (예정)
- [ ] 사용자 취향 기반 맞춤 추천
- [ ] 알레르기 정보 필터링
- [ ] 상황별 조건 설정 (주차, 음식 종류 등)

## 기술 스택
- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
